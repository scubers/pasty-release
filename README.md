# Pasty 使用指南

> 当前文档以 macOS 版本为准，目标是帮助你在几分钟内上手 Pasty，并逐步发现更高效的使用方式。

## Pasty 是怎么用的

把 Pasty 理解成一个始终待命的剪贴板工作台：

- 你像平时一样复制内容
- 需要回看、筛选、复用时，用 `⌘⇧V` 打开主面板
- 选中一条记录后，按 `Enter` 直接粘贴回当前应用
- 如果只是想重新复制，不想立刻粘贴，按 `⌘↩`

它的核心卖点不是“存很多条历史”，而是让你尽量不用鼠标、尽量不断流地完成复制、筛选、整理和再次使用。

## 第一次安装后先做什么

第一次启动时，Pasty 会优先引导你完成“辅助功能”授权。这个权限的用途很单一：当你在 Pasty 里选中一条历史记录后，Pasty 需要把“粘贴”这个动作发回你当前正在使用的应用。

你会看到欢迎页，里面主要有两个动作：

- `请求授权`
- `打开系统设置`

如果系统没有自动跳转，可以手动前往：

`系统设置 > 隐私与安全性 > 辅助功能`

然后把 Pasty 打开。

如果你当时先跳过了，也不用重装。之后仍然可以通过 `设置 > 通用 > 辅助功能权限` 回来补开。

## 三个最重要的入口

### 1. `⌘⇧V`：打开主面板

这是最常用的入口。默认情况下，`⌘⇧V` 会呼出 Pasty 主面板，用来浏览最近复制过的内容、搜索、筛选、打标签，或者再次粘贴。

这个快捷键可以在设置里改掉：

`设置 > 通用 > 快捷键`

### 2. `⌘,`：打开设置

当 Pasty 在前台时，可以用 `⌘,` 打开设置。这里可以调整快捷键、开机启动、语言、剪贴板采集过滤、检测能力、云同步等配置。

如果你记不住这组快捷键，也可以从菜单栏图标进入设置。

### 3. 菜单栏图标：最稳妥的入口

Pasty 会常驻菜单栏。点击菜单栏图标后，通常会看到这些入口：

- `打开面板`
- `设置...`
- `检查更新...`
- `退出`

如果你暂时不想记快捷键，或者刚完成安装，直接从菜单栏图标开始用就够了。

## 3 分钟快速上手

### 第一步：先像平时一样复制

无论是文本、图片，还是文件/文件夹引用，你都可以先按自己原本的习惯复制。Pasty 会把这些内容记录到历史里。

### 第二步：用主面板找回它

按 `⌘⇧V` 打开主面板后，你可以直接做两件事：

- 不输入任何内容，直接浏览最近的剪贴板历史
- 直接输入关键词搜索

如果你已经看到目标内容，选中它后：

- `Enter`：直接粘贴到当前应用
- `⌘↩`：重新复制这条内容，但不立即粘贴

这是最基本、也最高频的工作流。

### 第三步：开始用搜索，而不是靠回忆

当历史越来越多时，不要回忆“我是几分钟前复制的”，直接搜索更快。

主面板支持按关键词找内容，也支持按类型缩小范围。当前高频筛选包括：

- 文本
- 图片
- 文件
- 链接
- 标签
- OCR

最实用的方式有两种：

- 直接输入关键词
- 用 `⌃Tab` 在常用类型筛选之间快速切换

如果搜索框里已经有查询文本，切换筛选时不会把你的输入清掉。

## 主面板里最常用的操作

### 回贴与复用

选中一条历史记录后：

- `Enter`：粘贴
- `⌘↩`：复制

如果你经常需要把几分钟前、几小时前、甚至昨天复制过的内容再用一次，这两个动作会是最常见的组合。

### 标签

如果某些内容会反复用到，比如测试账号、常用命令、固定话术、项目链接，可以给它们打标签。

选中记录后按 `⌘T`：

- 直接输入标签名
- 有建议时可以用 `Tab` 选择
- 按 `Enter` 保存

当你以后搜索标签时，会比纯文本搜索更稳定。

### 固定

不希望某条记录被轻易淹没时，可以按 `⌘P` 固定或取消固定。适合长期保留的内容，比如常用地址、模板文案、内部环境信息。

### 删除

不再需要的记录可以按 `⌘D` 删除。删除前会有确认提示。

## 键盘党最该掌握的一件事：长按 `Cmd`

Pasty 的高频操作很多，但你不用硬背。

在主面板里，先按住 `Cmd` 不放，界面会根据你当前所在的位置，把可用快捷键直接显示出来。你可以把它理解成“上下文快捷键图层”：

- 在主面板列表里，会显示当前条目可执行的动作
- 在搜索框附近，会显示和搜索有关的快捷键
- 在检测结果区，会显示对应动作的数字提示
- 在标签编辑、删除确认这类局部界面里，也会切换成该界面的局部快捷键

如果你第一次接触 Pasty，最好的学习方式不是看完整快捷键列表，而是：

1. 打开主面板
2. 选中一条记录
3. 按住 `Cmd`
4. 看见什么，就按什么

这会比背文档快很多。

## 检测结果区怎么用

当 Pasty 识别到某条内容里包含结构化信息时，主面板下方会出现检测结果区。当前常见的识别内容包括：

- 链接
- OCR 文本
- JSON
- 二维码内容
- 颜色值
- 文件路径/文件引用信息

这一区域不是“附加说明”，而是可直接操作的工作区。你可以在里面：

- 复制识别结果
- 打开链接
- 显示二维码
- 打开详情
- 在 Finder 中定位文件
- 直接打开文件路径

高频键盘操作：

- `⌘⇧P`：展开或折叠检测结果区
- `⌃←` / `⌃→`：切换不同检测结果
- 按住 `Cmd`：显示当前动作的数字提示
- `⌘1` 到 `⌘9`：直接执行对应动作

如果你经常复制链接、截图、代码片段或文件路径，这个区域会非常有用。

## Action Panel：当“复制回来”还不够时

如果你选中了一条记录，按 `⌘K` 可以打开 Action Panel。它适合“不是简单复用，而是要基于当前内容做一次处理”的场景。

当前内置能力包括：

- URL 编码 / 解码
- Base64 编码 / 解码
- JSON 转义 / 反转义
- MD5 / SHA256 / SHA512
- 生成二维码
- 图片复制为 PNG / JPEG
- 图片压缩
- 另存为文件

一个简单判断方式是：

- 只是想再次使用原内容：用 `Enter` 或 `⌘↩`
- 想把当前内容先转换、导出或加工：用 `⌘K`

## 搜索不止是输入关键词

当你已经熟悉基础搜索后，可以再往前一步：

- 用 `⌃Tab` 在常用类型之间轮换
- 在搜索里输入类似 `tag:`、`file:` 这样的前缀，然后按 `Tab`，把它补齐成筛选 chip
- 删除筛选时，输入框为空后按两次 `Backspace`，可以按顺序选中并删除前面的 chip

这套方式的价值在于：搜索词和筛选条件可以一起保留，不需要每次重新组织查询。

## 设置里值得优先看的几项

第一次熟悉应用时，建议优先看这些设置：

- `通用 > 快捷键`：修改主面板唤起快捷键
- `通用 > 辅助功能权限`：如果第一次跳过了授权，可以在这里补开
- `通用`：开机启动、语言、主面板搜索恢复等
- `剪贴板`：配置不想被采集的来源应用
- `检测`：调整检测相关能力
- `云同步`：如果你明确需要跨设备同步，再在这里启用

建议先把本地工作流跑顺，再考虑同步和更高级的配置。

## 你可以把 Pasty 当成哪些能力在用

如果用一句话概括，Pasty 当前最适合承担这几类任务：

- 剪贴板历史：找回你复制过的文本、图片、文件和文件夹引用
- 键盘检索：通过关键词、类型和标签快速定位内容
- 快速复用：一键重新复制，或直接回贴到当前应用
- 内容整理：用标签和固定整理高价值记录
- 内容理解：对链接、OCR、JSON、二维码、颜色、路径等结果做结构化识别
- 内容处理：通过 Action Panel 对现有内容做转换、导出或派生操作

## 高频快捷键对照表

| 快捷键 | 用途 |
| --- | --- |
| `⌘⇧V` | 打开或切换主面板 |
| `⌘,` | 打开设置 |
| `Enter` | 将当前选中记录直接粘贴回当前应用 |
| `⌘↩` | 重新复制当前选中记录 |
| `⌘T` | 编辑标签 |
| `⌘P` | 固定 / 取消固定 |
| `⌘D` | 删除当前记录 |
| `⌘K` | 打开 Action Panel |
| `⌃Tab` | 在常用类型筛选之间切换 |
| `Tab` | 在搜索或标签场景中补齐 / 选择建议 |
| 长按 `Cmd` | 显示当前上下文可用快捷键 |
| `⌘⇧P` | 展开 / 折叠检测结果区 |
| `⌃←` / `⌃→` | 在检测结果之间切换 |
| `⌘1` ... `⌘9` | 执行检测结果区中对应编号的动作 |
| `Esc` | 关闭当前局部界面，或取消某些操作 |

## 推荐的上手顺序

如果你只想花最少时间把 Pasty 用起来，建议按这个顺序熟悉：

1. 先完成辅助功能授权
2. 记住 `⌘⇧V`
3. 学会 `Enter` 和 `⌘↩`
4. 学会搜索和 `⌃Tab`
5. 学会长按 `Cmd` 看提示
6. 再开始用 `⌘T`、`⌘P`、`⌘K`

当你已经能稳定使用第 1 到第 5 步时，Pasty 的大部分价值就已经出来了。
